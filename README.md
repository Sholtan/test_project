# test_project

Проект реализует простой прототип API на FastAPI с одним эндпоинтом /predict, который умеет выполнять две разные задачи. 

Первая задача: 
1) распознавание автомобильного номера: YOLO используется для детекции области номерного знака, затем EasyOCR извлекает оттуда текст; эта выполняется в модуле simple_ocr.
2) распознавание номеров контейнеров: первая модель YOLO находит бокс с номером и вторая модель YOLO уже детектит отдельные буквы и номера.


Вторая задача — прогноз временного ряда с помощью линейной регрессии, реализованная в файле time_series_model.py через функцию forecast_time_series(). 

В файле api.py я объединил обе функции и добавил выбор режима работы (mode: "ocr" или "ts"), чтобы API возвращал результат в формате JSON.
